Java 中一些不常用的函数

发表于 2018-07-26 更新于 2024-07-28 阅读次数： Waline：

对于我来说目前比较不怎么常用的函数

字符串和数字转换

数字转字符串

字符串转数字

浮点型（不能有字母）

数学

四舍五入，随机数，开方，次方，π，自然常数

求一千万以内的质数个数

字符串控制

charAt (int index) 获取指定位置的字符

toCharArray () 获取对应的字符数组

subString 截取子字符串

split 根据分隔符进行分隔

trim 去掉首尾空格

toLowerCase 全部变成小写

toUpperCase 全部变成大写

定位

indexOf 判断字符或者子字符串出现的位置
contains 是否包含子字符串

异常

finally

finally里一般拿来做一些善后清理工作,try块里出现错误的话，会立即跳出try块，找到匹配的错误，执行catch块里的语句此时，可能在try块里打开的文件没关闭，连接的网络没断开，对这些浪费的内存就不能及时释放回收。如果有finally块的话，不管有没有出错，都会执行finally块里的内容。

无论是否出现异常，finally 中的代码都会被执行代码比较复制代码
