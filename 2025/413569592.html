​解决办法简述

使用 ctrl+shift+f10

前言：

自从更新 win11 之后，总是会出现文件夹打开无响应，重启后电脑黑屏只有鼠标的情况。某度和其他平台的不管是图文，还是视频都说使用 ctrl+shift+del 快捷键调出任务管理器解决。但是关于使用快捷键后任务管理器无法显示的问题，都未给出解决办法。这里提供一个误打误撞调出显示界面的方式：使用 ctrl+shift+f10。下面是具体说明

电脑及 win11 版本介绍

电脑

win11 版本

描述及解决方案：

开机电脑黑屏，只有能看到鼠标指针。

使用 ctrl+shift+del 快捷键尝试调出任务管理器，可以到看到任务管理器选项，但是点击后并不能显示出任务管理器，电脑还是黑屏只有鼠标。

尝试按 ctrl+shift+f10。桌面显示出来了，但是任务栏不正常。

再点击 ctrl+shift+f10，又黑屏只有鼠标，ctrl+shift+f10 可以在桌面和黑屏之间来回切换

评论区

以下为评论内容的精华部分：

@xpp2_

分享一下，今天也是遇到这样情况了，c+s+f10 之后可以显示桌面之后调出任务管理器，找到 windows 资源管理器这个进程，右键重启进程就好了。偶尔遇到的任务栏卡死或者任务栏消失，都可以试试这样解决。

紧跟着更新一下：这种情况下，这种方法只是能出来任务栏，各种加载项和自启动好像都没正常加载。看了评论区有提到 win 菜单点睡眠再重进的方法，建议这样试试，重进之后各种加载项和开机自启动应用能正常出来了

@熊白白熊

确实能解决，我试出来 开机登录 - 黑屏 - ctrl alt del 出来界面后点右下角睡眠 - 再开机，也能进去。两个方法都挺恶心的。应该是用户文件哪里出问题了。新建一个用户登录就没问题。

@孔特里亚诺 m

按照这个方法做了还是只有桌面没有任务栏，结果把网线拔了就好了 [笑哭]

@别抢我的橘子 T_T

开机前可以先拔网线进去后，找服务，禁用 web 账户管理器，再连网重启试试

@队友快冲对面大残

楼主神医。我发现这种情况似乎是出现在我把 ipv4 的 ip 地址固定之后，然后出现这种情况之后进入桌面看到 IP 地址变成了 169.254.112.115，跟我设置的不一样，怀疑是 ip 地址冲突导致的
