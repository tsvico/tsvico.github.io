дҪҝз”Ё GitHub Actions иҮӘеҠЁйғЁзҪІ Hexo

еҸ‘иЎЁдәҺ 2021-03-07 жӣҙж–°дәҺ 2024-06-27 йҳ…иҜ»ж¬Ўж•°пјҡ Walineпјҡ

жңҖиҝ‘еҚҮзә§дәҶ next дё»йўҳеҲ° v8.2 зүҲжң¬пјҢжҗңзҙўеҗҺжүҚеҸ‘зҺ°жңҖж–°зҡ„ hexo-theme-next жӣҙжҚўд»“еә“еҲ° https://github.com/next-theme/hexo-theme-next, е№¶дё”ж”ҜжҢҒ npm ж–№ејҸйғЁзҪІпјҢиҖҢдёҚз”ЁеҶҚ pull жәҗз ҒпјҢдҪҝз”Ё npm е®үиЈ…д»ҘеҗҺеҚҮзә§еҸӘйңҖиҰҒ npm install hexo-theme-next@latest е°ұиЎҢдәҶгҖӮ
жӯЈеҘҪд»ҘеүҚз”ЁиҝҮ Azure DevOps е’Ң Travis CI зҡ„жү“еҢ…еҠҹиғҪпјҢе°ұжғізқҖиғҪдёҚиғҪз”Ёзұ»дјјзҡ„ж–№жі•зӣҙжҺҘйғЁзҪІеҲ° gitPageгҖҒcodingPageпјҢдәҺжҳҜжҗңзҙўдәҶзӣёе…ій—®йўҳгҖӮ

дҪҝз”Ё Actions еҸҜд»ҘеҫҲж–№дҫҝзҡ„еңЁ github дёҠеҜ№ж–Үз« иҝӣиЎҢдҝ®ж”№дёҺж–°еўһ

hexo-next зҡ„еҚҮзә§

д»Һ next v7->v8 иҝҳжҳҜжңүи®ёеӨҡең°ж–№дёҚдёҖж ·зҡ„пјҢжӣҙеҲ«иҜҙдҪҝз”ЁдәҶ npm зҡ„ж–№ејҸе®үиЈ…дё»йўҳгҖӮ

  • йҰ–е…ҲжҳҜеӨ–йғЁй…ҚзҪ®ж–Үд»¶пјҢеңЁ 7.x зүҲжң¬дёӯпјҢеӨ–йғЁй…ҚзҪ®ж–Үд»¶жҳҜж”ҫеңЁ source/_data/next.yml дёӯзҡ„пјҢиҖҢеңЁ 8.x зҡ„зүҲжң¬еҲҷжҳҜж”ҫеңЁдәҶйЎ№зӣ®ж №зӣ®еҪ•пјҢдёҺ_config.xml еҗҢзә§пјҢе№¶дё”е‘ҪеҗҚдёә_config.next.yml, е…·дҪ“е®үиЈ…еҸҜд»ҘжҹҘзңӢ next е®ҳж–№ж–ҮжЎЈпјҢйғҪжҜ”иҫғиҜҰз»Ҷ

  • v8 дёӯ valine дёҚеҶҚжҳҜеҶ…зҪ®иҜ„и®әжЁЎеқ—пјҢеҸҜиғҪиҝҷж¶үеҸҠеҲ°дёҖдәӣ xss жҲ–е…¶д»–ж–№йқўзҡ„е®үе…Ёй—®йўҳпјҢе…·дҪ“еҺҹеӣ дёҚжё…жҘҡпјҢдёҚиҝҮеҸҜд»ҘдҪҝз”Ё npm install next-theme/hexo-next-valine жүӢеҠЁе®үиЈ…пјҢеҶҚе°Ҷй…ҚзҪ®ж·»еҠ еҲ°_config.next.yml дёӯ

    еӣ дёәжәҗз Ғдёӯ hexo-next-valine дҪҝз”ЁдәҶ Object.assignпјҢжүҖд»ҘзҗҶи®әдёҠ valine е®ҳж–№жүҖжңүж”ҜжҢҒзҡ„еҸӮж•°еңЁиҝҷйҮҢйғҪеҸҜд»Ҙиў«ж”ҜжҢҒпјҢеҢ…жӢ¬ emojiCDN зӯүзӯүгҖӮ

з”ҹжҲҗеҜҶй’Ҙ

е…ідәҺеҰӮдҪ•е°Ҷ hexo еҗҢж—¶йғЁзҪІеҲ° GitHub Pages е’Ң coding Page иҝҷйҮҢдёҚеҶҚиөҳиҝ°еҸҜд»ҘеҸӮиҖғиҝҷйҮҢ 2018/c067e22a.htmlпјҢиҝҷйҮҢдҪ еҸҜд»Ҙз”ҹжҲҗдёҖдёӘж–°зҡ„еҜҶй’Ҙдё“й—Ёз”ЁдәҺ Actions йғЁзҪІпјҢд№ҹеҸҜд»ҘдҪҝз”ЁеҺҹжқҘзҡ„еҜҶй’Ҙж–Үд»¶

з”ҹжҲҗж–№жі•пјҢдҪҝз”Ё gitBash

дјҡеңЁ C:\Users\з”ЁжҲ·еҗҚ\.ssh зңӢеҲ°дёӨдёӘж–Үд»¶

  • github-deploy-key вҖ”вҖ” з§Ғй’Ҙ
  • github-deploy-key.pub вҖ”вҖ” е…¬й’Ҙ

й…ҚзҪ®з§ҳй’Ҙ

жҲ‘д»¬иҰҒеңЁ GitHub дёҺ Coding дёҠйғЁзҪІеҜҶй’ҘпјҢе…¶дёӯз§Ғй’Ҙз”ЁдәҺ Hexo жәҗз Ғд»“еә“йҮҢ пјҢжәҗз Ғд»“еә“жҲ‘д»¬еҸҜд»Ҙж–°е»әдёҖдёӘз©әзҡ„з§Ғжңүд»“еә“пјҢз„¶еҗҺе°ҶеҪ“еүҚзҡ„ж–Үд»¶жәҗз Ғе…іиҒ”еҲ°иҝҷдёӘд»“еә“дёӯпјҢеңЁжәҗз Ғзӣ®еҪ•жү§иЎҢ (жәҗз Ғзӣ®еҪ•жҢҮзҡ„жҳҜеҢ…еҗ« package.json ж–Үд»¶зҡ„ hexo ж №зӣ®еҪ•)

жҠҠе…¬й’Ҙж”ҫеҲ° GitHub Pages е’Ң coding еҜ№еә”зҡ„д»Јз Ғд»“еә“йҮҢйқўпјҢз”ЁдәҺ Hexo йғЁзҪІж—¶зҡ„еҶҷе…Ҙе·ҘдҪң

й…ҚзҪ®з§Ғй’Ҙ

йҰ–е…ҲеңЁ GitHub дёҠжү“ејҖдҝқеӯҳ Hexo зҡ„д»“еә“пјҢи®ҝй—® Settings -> SecretsпјҢз”»йқўеҰӮдёӢпјҡ

з„¶еҗҺйҖүжӢ© New secret

еҗҚеӯ—йғЁеҲҶеЎ«еҶҷпјҡHEXO_DEPLOY_KEYпјҢжіЁж„ҸеӨ§е°ҸеҶҷпјҢиҝҷдёӘеҗҺйқўзҡ„ GitHub Actions Workflow иҰҒз”ЁеҲ°пјҢдёҖе®ҡдёҚиғҪеҶҷй”ҷгҖӮ

еңЁ Value зҡ„йғЁеҲҶеЎ«е…Ҙ github-deploy-key дёӯзҡ„еҶ…е®№пјҡ

ж·»еҠ дәҶз§Ғй’Ҙд»ҘеҗҺзҡ„з•ҢйқўжҳҫзӨәеҰӮдёӢпјҡ

ж·»еҠ е…¬й’Ҙ

github Page

жҺҘдёӢжқҘжҲ‘д»¬йңҖиҰҒи®ҝй—®еӯҳж”ҫзҪ‘йЎөзҡ„д»“еә“пјҢд№ҹе°ұжҳҜ Hexo йғЁзҪІд»ҘеҗҺзҡ„д»“еә“пјҢжҜ”еҰӮпјҡxxx.github.io иҝҷз§ҚпјҢи®ҝй—® Settings -> Deploy keysпјҡ

жҢү Add deploy key жқҘж·»еҠ дёҖдёӘж–°зҡ„е…¬й’Ҙпјҡ

еңЁ Title дёӯиҫ“е…ҘпјҡHEXO_DEPLOY_PUB еӯ—ж ·пјҢеҪ“з„¶д№ҹеҸҜд»ҘеЎ«еҶҷе…¶е®ғиҮӘе®ҡд№үзҡ„еҗҚеӯ—гҖӮ

еңЁ Key дёӯзІҳиҙҙ github-deploy-key.pub ж–Үд»¶зҡ„еҶ…е®№гҖӮ

жіЁж„ҸпјҡдёҖе®ҡиҰҒеӢҫйҖү Allow write access жқҘжү“ејҖеҶҷжқғйҷҗпјҢеҗҰеҲҷж— жі•еҶҷе…ҘдјҡеҜјиҮҙйғЁзҪІеӨұиҙҘгҖӮ

жңҖеҗҺж·»еҠ еҘҪдәҶе…¬й’Ҙзҡ„з•ҢйқўеҰӮдёӢпјҡ

coding Page

жү“ејҖ д»Јз Ғд»“еә“->и®ҫзҪ®->йғЁзҪІе…¬й’ҘпјҢзӮ№еҮ»ж–°е»ә

еңЁ Title дёӯиҫ“е…ҘпјҡHEXO_DEPLOY_PUB еӯ—ж ·пјҢеҪ“з„¶д№ҹеҸҜд»ҘеЎ«еҶҷе…¶е®ғиҮӘе®ҡд№үзҡ„еҗҚеӯ—гҖӮ

еңЁ Key дёӯзІҳиҙҙ github-deploy-key.pub ж–Үд»¶зҡ„еҶ…е®№гҖӮеҗҢж ·зҡ„пјҢжіЁж„ҸеӢҫйҖүжҺҲдәҲжҺЁйҖҒжқғйҷҗе’Ңе…¬й’Ҙж°ёд№…жңүж•Ҳжңҹ

ж·»еҠ е®Ңзҡ„ж•ҲжһңеҰӮдёӢ

еҲӣе»ә Workflow

йҰ–е…ҲеңЁ Hexo зҡ„д»“еә“дёӯеҲӣе»әдёҖдёӘж–°ж–Үд»¶пјҡ.github/workflows/deploy.ymlпјҢж–Үд»¶еҗҚеҸҜд»ҘиҮӘе·ұеҸ–пјҢдҪҶжҳҜдёҖе®ҡиҰҒж”ҫеңЁ .github/workflows зӣ®еҪ•дёӯпјҢж–Үд»¶зҡ„еҶ…е®№еҰӮдёӢпјҡ

з®ҖеҚ•и§ЈйҮҠдёҖдёӢпјҢеҪ“жҲ‘д»¬жҺЁйҖҒеҶ…е®№еҲ°иҝңзЁӢ master еҲҶж”Ҝзҡ„ж—¶еҖҷпјҢе°ұдјҡи§ҰеҸ‘иҝҷдёӘ WorkflowпјҢеҗҢж—¶з»ҷиҮӘе·ұйЎ№зӣ®зӮ№еҮ» star д№ҹдјҡи§ҰеҸ‘гҖӮ

дҪҝз”Ё Ubuntu дҪңдёә hexo deploy зҡ„зі»з»ҹгҖӮ

йҰ–е…Ҳ checkout жәҗд»Јз ҒпјҢз„¶еҗҺи®ҫзҪ®дҪҝз”ЁжңҖж–°зҡ„ Node.js v12 LTS дҪңдёә node и§ЈйҮҠеҷЁгҖӮ

жҺҘдёӢжқҘе°ұжҳҜеҲӣе»ә SSH зӣёе…ізҡ„й…ҚзҪ®ж–Үд»¶пјҢжіЁж„Ҹ secrets.HEXO_DEPLOY_KEY е°ұжҳҜеҜ№еә”жҲ‘д»¬д№ӢеүҚи®ҫзҪ®зҡ„з§Ғй’ҘпјҢжүҖд»ҘеҗҚеӯ—дёҖе®ҡдёҚиҰҒжҗһй”ҷгҖӮ

git config зӣёе…ізҡ„еҗҚеӯ—е’ҢйӮ®д»¶ең°еқҖжӣҝжҚўжҲҗиҮӘе·ұдҪҝз”Ёзҡ„гҖӮ

жңҖеҗҺе°ұжҳҜе®үиЈ… Hexo CLIпјҢеҗ„дёӘдҫқиө–жЁЎеқ—е’ҢйғЁзҪІдәҶгҖӮ

йӘҢиҜҒ

дёӢйқўе°ұжҳҜ GitHub Actions йЎөйқўжҳҫзӨәзҡ„иҝҗиЎҢз»“жһңпјҡ

з»ҝиүІзҡ„пјҢе°ұиЎЁзӨәйғЁзҪІжҲҗеҠҹпјҢзәўиүІзҡ„иЎЁзӨәеӨұиҙҘгҖӮеҰӮжһңйғЁзҪІеӨұиҙҘпјҢеҸҜд»ҘзӮ№еҮ»йғЁзҪІд»»еҠЎжҹҘзңӢй”ҷиҜҜж—Ҙеҝ—пјҢиҝӣиЎҢи°ғиҜ•пјҢзӮ№еҮ»йЎ№зӣ® star еҸҜд»ҘиҝӣиЎҢжүӢеҠЁйғЁзҪІгҖӮ
